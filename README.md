# afd-iac
IaC with TF for AFD


## Steps
* Create a new conda environment
* Install Terraform
* Install Azure CLI
* Authenticate with az login
* Set subscription
* Create a service principal
* Set environment variables with Client ID, Client Secret, Subscription ID & Tenanat ID

## Configuration Changes
* terraform validate
* terraform fmt
* terraform plan
* terraform apply
